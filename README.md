# Commerce OS

Next.js storefront starter for the e-commerce project:

- Next.js App Router
- React 19
- Tailwind CSS 4
- Supabase SSR/client helpers
- Lucide icons

## Commands

```bash
npm run dev
npm run build
```

## Structure

- `app/page.tsx` - storefront, product grid, cart preview, operations panel
- `lib/storefront.ts` - starter product data and money formatting
- `lib/supabase` - browser/server Supabase client helpers
